# file-viewer [![Build Status](https://travis-ci.org/sgkim126/file-viewer.svg?branch=master)](https://travis-ci.org/sgkim126/file-viewer)

## Dependencies
* Python 3.5
    * tornado
    * webpack
* Node 4.x
    * typescript
    * stylus
    * babel

## Run
```
make run
```
or
```
make run PORT={port}
```
